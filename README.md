A mobile app to create and share surprise boxes. Add text, images, dates, locations, word games and puzzles — each hidden behind an unlock code. Share a unique code with someone and let them discover your surprises one by one.

## Features

- Create surprises with multiple elements (text, image, date, location, word game, puzzle)
- Share via a unique 6-character code or deep link
- Unlock elements with a code
- Customizable theme color per surprise
- Multi-language support (English, French, Spanish)

## Tech Stack

- Flutter (iOS & Android)
- Supabase (backend & storage)
- Provider (state management)

## Getting Started

```bash
flutter pub get
flutter run
```
