Surprise Me

Politique de confidentialité

📋 En résumé

Surprise Me ne collecte aucune donnée personnelle identifiable. L'application ne nécessite pas de création de compte, ne suit pas votre comportement et ne contient aucune publicité.

✓ Aucun compte requis  ✓ Aucune publicité  ✓ Aucun tracking

☁️ Données stockées sur nos serveurs

Les informations suivantes sont enregistrées dans notre base de données afin de faire fonctionner l'application :

  • Le contenu des surprises que vous créez : titre, sous-titre, emoji, couleur, éléments (texte, dates, lieux, images, jeux).
  • Un code de partage aléatoire permettant à d'autres utilisateurs de rejoindre votre surprise.
  • Un token de création (UUID aléatoire) permettant de vérifier que vous êtes bien l'auteur de la surprise.
  • Les images que vous uploadez, hébergées sur notre espace de stockage sécurisé.

Ces données ne contiennent aucun nom, email, numéro de téléphone ou identifiant personnel.

📱 Données stockées localement sur votre appareil

Les informations suivantes restent exclusivement sur votre appareil et ne sont jamais envoyées :

  • Les codes des surprises que vous avez créées ou rejointes.
  • Les codes de déverrouillage des éléments que vous avez révélés.
  • Le token de création de vos surprises (pour vous permettre de les modifier).

📷 Accès à la caméra et à la galerie

L'application peut demander l'accès à votre appareil photo et à votre galerie uniquement lorsque vous ajoutez une image à une surprise. Ces permissions sont optionnelles et ne sont utilisées qu'à votre demande explicite.

⚠ Aucune photo n'est collectée sans votre action

🔗 Services tiers

L'application utilise les services suivants :

  • Supabase – base de données et stockage des fichiers. Vos données sont hébergées sur des serveurs sécurisés. Politique de confidentialité Supabase →

Aucun autre service tiers (analytics, publicité, réseaux sociaux) n'est intégré.

🗑️ Suppression des données

Vous pouvez supprimer vos surprises directement depuis l'application. Cette action supprime définitivement toutes les données associées sur nos serveurs.

Pour toute demande de suppression de données résiduelles, contactez-nous à l'adresse ci-dessous.
